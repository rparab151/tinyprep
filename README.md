# TinyPrep

TinyPrep is a free toddler meal-prep app prototype for busy parents. It helps plan weekly toddler meals, batch prep freezer-friendly foods, track accepted/trying/retry-later meals, and generate a deduplicated grocery list.

## Current state

This repository currently contains a production-shaped React/Vite prototype. The next step is to convert it into an Expo React Native app for iOS App Store and Google Play Store submission.

## Run locally

```bash
npm install
npm run dev
```

## Product principles

- Free app, no subscription paywall
- Offline-first where possible
- No ads in the toddler/parenting experience
- No medical or nutrition claims beyond general meal-prep guidance
- Clear allergy and choking-risk disclaimers before production release
